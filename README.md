## About This Project
This is my personal portfolio project built using **Laravel** and **Tailwind CSS**. It showcases my web development skills and demonstrates a clean, responsive design.

### Features
- Fully responsive layout using Tailwind CSS
- Laravel backend with structured routing and controllers
- Modern UI/UX design
- Deployment-ready configuration
